# Лео Считалкин 🦊

Математика для второклассника: сложение и вычитание до 100 и таблица умножения.
PWA — ставится на телефон и работает без интернета. Весь прогресс хранится
только на устройстве.

---

## Запуск

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # → dist/
npm run icons      # перерисовать иконки PWA из мордочки Лео
```

## Деплой

`vite.config.js` использует `base: './'`, поэтому **одна и та же сборка** работает
и в корне домена (Vercel), и в подпапке `/<repo>/` (GitHub Pages) — без
пересборки и без подстановки имени репозитория.

- **Vercel** — `vercel.json` уже настроен, автодеплой при push в `main`.
- **GitHub Pages** — `.github/workflows/deploy.yml`, автодеплой при push в `main`.
  Включите Pages → Source: **GitHub Actions**.

Роутинг на `HashRouter`, так что серверные rewrites не нужны нигде.

## Как это устроено

```
src/
├── components/     Leo (маскот), Confetti, ui (кнопки, сердечки, звёзды, наклейки)
├── exercises/      по одному файлу на тип задания
├── game/           curriculum, generators, store, audio, confetti, stickers
├── screens/        Home (карта), Lesson, LessonComplete, Collection, Settings, Tutor
└── styles/         tokens.css (палитра, размеры), global.css
```

**Контент** описан в `game/curriculum.js` (два раздела, 18 узлов), а вопросы
генерируются на лету в `game/generators.js`. Добавить урок = добавить запись в
curriculum + ветку в `buildLesson`.

**Прогресс** — `game/store.jsx`, React Context + reducer, пишется в
`localStorage` под ключом `leo-schitalkin/v1`. Ничего не уходит в сеть.

**Звук** синтезируется через WebAudio (`game/audio.js`) — ноль байт загрузки,
мгновенный отклик, работает офлайн. Озвучка заданий — Web Speech API.

**Шрифт** Nunito (SIL OFL) лежит в `src/assets/fonts/` — подмножества latin +
cyrillic, 60 КБ, чтобы офлайн выглядело так же, как онлайн.

### Педагогика, которая зашита в код

- Порядок таблиц — **2, 10, 5, 3, 4, 6, 9, 7, 8**, а не по возрастанию: каждая
  следующая опирается на уже освоенную (×4 — это ×2 дважды, ×9 — это ×10 минус
  число). ×7 и ×8 в конце, потому что к тому моменту почти всё в них уже
  выучено как перестановки.
- **Точность считается по первой попытке.** Правильный ответ со второй даёт XP,
  но не влияет на звёзды — иначе метрика «что даётся тяжело» врёт.
- «Смешанная тренировка» и «Блиц» тянут факты с **весом по ошибкам**
  (`weightedFactPool`): то, что не получается, встречается чаще.
- Ошибка **никогда не заканчивает урок**. Сердечки только показывают старание.
  Цвет ошибки — тёплый оранжевый, звук — мягкий, Лео — озадаченный, но добрый.

## Дев-инструменты (`scripts/`)

```bash
node scripts/shot.mjs "#/" home 430 932 1500          # скриншот экрана
node scripts/play.mjs numberline done-nline           # пройти урок целиком
node scripts/probe.mjs "#/" "document.title"          # выполнить JS на странице
```

`play.mjs` реально проходит урок (отвечает на все вопросы, проверяет XP,
звёзды, наклейки, точность) — этим были найдены, например, потерянные нажатия
при быстром тапе. Он опирается на `window.__leoQ`, который есть только в dev
и вырезается из production-сборки.

## Родительская страница

`/#/tutor` — за шестерёнкой в углу и подтверждением «Я взрослый». Показывает
точность по темам, флаг «тяжело даётся» (ниже 60%), минуты за 2 недели и сетку
всех фактов таблицы умножения по цветам.
